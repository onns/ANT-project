# 关于Chrome中共享事件循环的时间攻击的漏洞

## 摘要

事件驱动编程很容易被监听，论文展示了针对Chrome的两个核心事件循环的攻击：I/O线程（用于支持所有网络事件和用户活动）和渲染进程的主线程（用于渲染和Javascript任务）。
